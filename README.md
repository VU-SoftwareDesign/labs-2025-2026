# JavaFX Observer
Example JavaFX application showing off the Observer design pattern. Made for the Software Design course at VU Amsterdam.

## Setup

1. Clone this branch into IntelliJ IDEA.
2. Go to Services (Alt+8) and create a new Run Configuration (+ button) and select **Maven**.
3. Add the following configurations:
    * **Run**: `clean javafx:run`.
    * **Working Directory**: This directory (probably `<YOUR_USER_FOLDER>/IdeaProjects/labs-2025-2026`).
4. Run! You should see a window pop up with a text box that responds to two text inputs.
